# workspace-machine
My local machine everywhere (using just Docker)

### Installed tools
- git
- vim

#### Future
- Java 9
- Clojure


### Docker

#### Build

```
docker build -t andreformento/workspace-machine .
```

#### Run and attach

The order is -> `host:container`

```
docker run \
       -it -P \
       -v $(pwd):/home \
       -e USERID=$UID \
       andreformento/workspace-machine /bin/bash
```
